Rating in Stars - Razor Component for Blazor.

- Allows users to select a rating in stars.
- Can also be used as fixed element on a page.
- Up to 5 stars. It is possible to select a half star.
- Uses Blazor/C#, HTML and CSS. No JavaScript needed.
- Change the colors, margins, sizes of the stars.
- Use custom PNG-images.
- Output of code & installation instructions.
